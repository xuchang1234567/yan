# 具身智能考研复习

面向 2029 级硕士招生的个人复习仓库，主线为 408，目标方向为具身智能与机器人学习。

## 当前决定

| 项目 | 决定 |
|---|---|
| 当前年级 | 2026 年秋季进入大二 |
| 预计初试 | 2028 年 12 月 |
| 预计入学 | 2029 年 9 月 |
| 专业课主线 | 408 计算机学科专业基础 |
| 公共课策略 | 前期按数学一、英语一能力要求准备，报名后按目标专业实际科目收口 |
| 目标方向 | 具身智能、机器人学习、VLA，保留控制基础 |

招生科目每年可能变化。本仓库中的院校信息只作为阶段性快照，最终以 2029 年招生目录和当年院系公告为准。

## 从这里开始

1. 立即执行 [2026 年暑假启动计划](plans/2026-summer.md)，先完成能力摸底。
2. 阅读 [总路线](docs/roadmap.md)，了解四层能力结构和完整时间轴。
3. 开学后执行 [2026 年秋季计划](plans/2026-fall.md)，每周只推进列出的重点。
4. 查看 [目标院校与考试科目](docs/schools.md)，不要用过期科目决定复习路线。
5. 每周复制 [周复盘模板](templates/weekly-review.md)，记录完成证据和下周任务。

## 记录规则

- 招生信息优先引用学校研招网、院系公告和教育部文件，并写明核验日期。
- 学习记录写“完成了什么”和对应证据，不用学习时长代替结果。
- 项目以可运行代码、实验记录、演示视频和技术说明为成果。
- 行业薪资、岗位增幅和融资数字只作背景，不作为选专业的主要依据。
