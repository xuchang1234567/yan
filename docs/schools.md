# 目标院校与考试科目

> 核验日期：2026-07-21。以下是 2026 目录和已发布的 2027 调整公告，不代表 2029 招生科目。

## 408 主线项目

| 学校与项目 | 当前可确认科目 | 当前判断 |
|---|---|---|
| 复旦 307，085400 电子信息 05-07 | 2026：政治、英二、数二、408 | 方向名称直接包含具身智能、感知和机器人硬件；新学院，需重点核验导师与统考计划 |
| 复旦 305，085400 电子信息 | 2026：政治、英二、数二、408 | 招生体量较大、AI/计算机覆盖广，可作为复旦校内另一选择 |
| 清华计算机系，085400 电子信息 | 2027 起：第四科改为 408 | 只确认相关专硕调整，不能写成清华所有计算机项目均考 408 |
| 清华深圳国际研究生院相关方向 | 2027 起：AI、大模型、计算机技术、大数据改为 408 | 与 VLA 基础匹配，但方向名称不等于具体导师一定做具身智能 |
| 上交 531 人工智能学院 | 2026：政治、英一、数一、408 | 总计划 45、推免计划 25；竞争风险高，导师匹配优先 |
| 上交 033 计算机学院 | 2026：相关统考方向为政治、英一、数一、408 | 计算机基础强，具身密度需按导师核验 |
| 深大 110，085410 人工智能 | 2026：政治、英一、数一、408 | 拟招 54，学院覆盖智能机器人等研究内容 |
| 深大 167，085410 人工智能 | 2026：政治、英一、数一、408 | 拟招 72，学院公开介绍包含具身智能研究中心 |

## 已纠正的错误

- 清华是部分相关项目进入 408 阵营，不是“清华系全面 408”。
- 复旦 307 的 2026 官方方向确有“多模态大模型与具身智能”等名称。
- 上交机械与动力工程学院 2026 年 085500 考 `813 机械设计原理与基础`，不是 809。
- “按数一英一准备”是学习范围策略，不是报名和考试科目的上下兼容规则。
- “39 所 985 全面进入 408”和精确行业薪资数字不作为选校依据。
