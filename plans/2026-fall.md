# 2026 年秋季执行计划

## 学期目标

本学期不进行完整 408 冲刺。目标是把课内成绩、数学、英语和编程地基打牢，为大三系统复习留下空间。

学期结束时应完成：

- 核心数学和计算机课程争取 85+，保留完整课堂笔记与错题。
- 如果尚未通过四级，本学期通过四级；已通过则开始六级准备。
- Python 能完成数据处理、实验绘图和一个小型项目。
- C++ 掌握基本语法、调试、STL，并能实现常见数据结构。
- 数据结构覆盖复杂度、线性表、栈、队列、树、图、查找和排序。
- 建立周复盘习惯，连续记录至少 12 周。

## 每周时间分配

课外学习建议从每周 8-10 小时起步，根据课内压力调整：

| 模块 | 建议时间 | 结果要求 |
|---|---:|---|
| 课内数学与错题 | 3-4 小时 | 当周知识点复述、作业订正、错题回做 |
| Python/C++/数据结构 | 3 小时 | 至少提交一次可运行代码 |
| 英语 | 2 小时 | 单词复习、阅读或四六级专项有可检查记录 |
| 周复盘与仓库整理 | 1 小时 | 更新复盘、清理下周任务 |

考试周优先保证课内复习，项目暂停不算失败。

## 四个阶段

### 第 1-4 周：建立系统

- 记录本学期课程、考试构成和目标分数。
- 完成 Python 与 C++ 基础自测，确定薄弱项。
- 复习复杂度、数组、链表、栈和队列。
- 建立英语词汇与阅读记录。
- 每周日完成一次复盘。

### 第 5-8 周：数据结构主干

- 学习树、二叉树、堆和哈夫曼树。
- 用 C++ 独立实现二叉树遍历、堆和优先队列。
- Python 完成一次 CSV 数据清洗与可视化。
- 完成英语阶段自测，调整四六级训练重点。

### 第 9-12 周：图与算法

- 学习图的存储、遍历、最短路和最小生成树。
- 补离散数学中的集合、关系、逻辑、图论基础。
- 完成一个命令行小项目，必须包含测试和 README。
- 整理前 12 周错题，找出重复错误。

### 第 13-16 周：课程收口

- 以课内考试和课程项目为最高优先级。
- 回顾查找、排序和全部数据结构复杂度。
- 形成一份数据结构知识图谱或章节索引。
- 汇总学期成果、成绩、代码和下一学期薄弱点。
